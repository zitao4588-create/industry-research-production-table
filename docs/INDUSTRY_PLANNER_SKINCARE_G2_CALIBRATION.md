# Industry Planner 护肤品 G2 校准记录

更新时间：2026-07-12
适用坐标：护肤品 / 中国大陆 / 2024–2026
权限与数据边界：L1/L2；仅使用无登录、无 cookie、无 API key、无 credits 的公开官方页面；本记录不是完整行业报告。

## 1. 校准结论

G2 保留 24 个规划项、18 类来源角色和 11 行覆盖矩阵，但修正了它们的语义：

- 24 个规划项是研究维度，不是 24 条行业事实。状态分为 8 个 `authority_aligned`、3 个 `method_guardrail`、13 个 `requires_live_validation`。
- 原 5 个 taxonomy 项混用了产品子类、功效和人群，存在重叠。现改为中国化妆品分类规则的五个正交维度：功效宣称、作用部位、产品剂型、使用人群、使用方法。
- 商业子市场不再冒充监管分类。清洁、保湿、防晒等可在后续作为功效组合或商业子市场候选，但必须由真实来源和样本另行验证。
- 原“大众 / 中高端 / 高端奢华”价格带预设了品牌语义，现改为可比单位价的低位、中位、高位组；数值边界必须在代表样本阶段按功效、剂型、规格和渠道校准。
- 国家统计局“化妆品类”包含护肤之外的洗发、护发等商品，不能直接支持护肤品市场规模或增速。
- 渠道、消费者需求和商业模式保留为研究假设；官方零售分类只支持如何拆渠道，不支持护肤品渠道份额。

## 2. 24 个规划项

| 轴 | 数量 | 校准后口径 | 状态 |
| --- | ---: | --- | --- |
| taxonomy | 5 | 功效宣称、作用部位、产品剂型、使用人群、使用方法 | authority aligned |
| value chain | 4 | 原料/配方/包装；研发/安全/检验/功效；注册备案/生产/受托加工；品牌/分销/零售 | 前 3 项 authority aligned；末项待验证 |
| price tier | 3 | 可比单位价低位、中位、高位；不预设商业标签或数值 | method guardrail |
| channel | 4 | 品牌直营端点、第三方平台线上零售、专业有店铺零售、综合有店铺零售 | requires live validation |
| consumer need | 4 | 功效结果、安全耐受、使用体验、信任决策 | requires live validation |
| business model | 4 | 品牌、自有/受托制造、零售平台、专业服务/内容 | requires live validation |

校准状态不等于事实验证。即使是 `authority_aligned`，Planner 也只确认研究维度与正式规则一致；具体产品、市场规模、需求强度和商业表现仍未采集。

## 3. 18 类来源角色

18 类角色全部保留，且每类现在都有：角色定义、允许支持的 claim role、最少证据要求、禁止外推和校准依据。

- `regulator` / `standards_body`：用于现行监管与标准，优先发布机关原文；不支持需求或机会。
- `government_statistics`：必须记录指标原名、地区、时间和范围，并核对统计类目与目标行业；不能跨口径外推。
- 协会、研究机构和财报：用于市场或公司层事实，必须披露方法/样本/主体边界，单一公司不能代表全行业。
- 品牌官网、官方店、可信零售：只支持品牌、产品、展示价格和渠道存在性，不能证明成交、需求、市场份额或利润。
- 评论、社区、用户研究、搜索趋势：只支持消费者信号，必须区分个案、重复信号和总体结论。
- 内容平台与创作者数据：只支持内容/流量信号，不能把曝光互动外推为转化。
- 公司材料、产业媒体和供应链公司：支持商业模式/供应链线索；盈利判断仍优先财报或等价证据。

## 4. 11 行覆盖目标

11 行不是统计样本量结论，而是三个 fail-closed 最低门槛：

1. `primary_authority_minimum`：监管问题至少绑定一个有管辖权的现行原文。一个来源是最低门槛，不代表完成法律审查。
2. `triangulation_guardrail`：市场 taxonomy、价格和渠道至少两个独立来源，用于暴露口径冲突，不宣称统计充分性。
3. `representative_sampling_guardrail`：消费者、竞品、内容、供应链和商业模式除交叉来源外还要跨轴样本，用于阻止单样本外推，不宣称统计代表性。

每行在 artifact 中记录 `targetBasis` 和 `calibrationRationale`。当前覆盖仍为 0 来源、0 角色、0 样本；G2 没有伪造覆盖完成。

## 5. 官方依据与适用限制

1. [《化妆品监督管理条例》（国务院令第727号）](https://www.beijing.gov.cn/zhengce/gwywj/202006/t20200630_1935123.html)：支持化妆品定义、四类分类因素、普通/特殊化妆品、注册备案和生产责任。限制：不支持市场规模、需求或渠道份额。
2. [国家药监局《化妆品分类规则和分类目录》（2021年第49号，福建省药监局官方转载全文）](https://yjj.scjgj.fujian.gov.cn/hzp/flfg/202106/t20210622_5630812.htm)：明确五维分类编码。限制：合规分类不等同商业市场分层。
3. [国家药监局政务服务门户：功效宣称评价办事口径](https://zwfw.nmpa.gov.cn/web/taskview/11100000MB0341032Y100017214900101)：支持功效评价与依据摘要核查问题。限制：具体产品仍需核对现行原文和备案资料。
4. [《化妆品注册备案管理办法》（市场监管总局令第35号）](https://www.samr.gov.cn/cms_files/filemanager/samr/www/samrnew/samrgkml/nsjg/fgs/202101/W020211127402192291034.pdf)：支持注册/备案边界。限制：注册备案不证明销售或需求。
5. [国家统计局关于“化妆品类”统计范围的答复](https://www.stats.gov.cn/hd/lyzx/zxgk/202405/t20240524_1954124.html)：确认该统计类目宽于护肤品。限制：未做口径转换前不得支持护肤品规模。
6. [国家统计局统计用零售业态分类答复](https://www.stats.gov.cn/hd/cjwtjd/202302/t20230207_1902274.html)：支持按经营方式、商品结构、服务功能和固定场所拆分渠道。限制：不支持护肤品渠道份额。
7. [国家统计局社会消费品零售总额定义与局限](https://www.stats.gov.cn/zs/tjws/tjzb/202301/t20230101_1903707.html)：说明总量指标不能充分反映消费结构，需要其他指标配合。

## 6. 保留待确认与下一阶段边界

- 本 fixture 的“中国大陆”是显式输入，不是系统默认市场，因此 G2 未替用户新增默认市场决策。
- 商业子市场命名、价格数值边界、代表性标准、具体平台清单和付费数据源仍未决定。
- G2 不启动完整护肤品报告、不调用 provider、不消耗 credits、不写生产、不 commit/push/deploy。
- 后续 G3 只能实现本地分阶段运行契约；真实来源发现和代表性抽样分别留给 G4、G5。
