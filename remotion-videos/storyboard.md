# 电商竞品研究 Agent · 产品介绍视频分镜

## 总览

- **产品**：电商竞品研究 Agent（行业研究生产台）
- **时长**：30 秒（30fps，共 900 帧）
- **尺寸**：1080×1920（抖音竖屏 9:16）
- **风格**：科技深色 + 数据流
- **目标**：30 秒讲清"输入行业品类 → 自动建九库 → 输出机会报告"的核心价值

## 设计系统

| 元素 | 取值 |
|------|------|
| 主背景 | `#0A0E27` → `#0F172A` 渐变 |
| 主强调色 | `#8B5CF6` 紫 / `#6366F1` 靛 |
| 次强调色 | `#22D3EE` 青 / `#38BDF8` 天蓝 |
| 成功/评分 | `#10B981` 翠绿 |
| 痛点/警告 | `#F43F5E` 玫红 |
| 主文字 | `#F8FAFC` 近白 |
| 次文字 | `#94A3B8` 石板灰 |
| 字体 | Space Grotesk（英文）+ Noto Sans SC（中文）粗体 |
| 数据流粒子 | 青紫色小点，从四周汇聚 / 流动 |

## 场景流

### 场景 1 · 痛点 Hook（0–4s，120 帧）
**文案**：做电商竞品调研，还在手动翻网页？

**视觉**：
- 深色背景，散乱的网页卡片碎片在画面飘动、错位
- 中部大字逐字浮现（typewriter），玫红强调"手动翻网页"
- 底部小字：「信息散落 · 难以复用 · 无法追踪」逐行淡入
- 收尾：碎片被一股吸力向中心收拢（暗示即将被整理）

### 场景 2 · 产品亮相（4–8s，120 帧）
**文案**：电商竞品研究 Agent / 输入行业品类，自动建库输出报告

**视觉**：
- 上场景收拢的碎片化作青紫数据流粒子，从四周汇聚到中心
- 粒子聚合成产品 Logo 图形（数据网格圆环 + 中心节点）
- 主标题 spring 弹出："电商竞品研究 Agent"
- 副标题淡入上滑："输入行业品类 · 自动建库 · 输出报告"
- 顶部小标签「行业研究生产台」+ 底部流动光带

### 场景 3 · 自动发现信息源（8–14s，180 帧）
**文案**：自动发现公开信息源 · 建立采集计划

**视觉**：
- 顶部输入框动画：依次填入「宠物食品」「益生菌」「北美市场」
- 下方流程图横向展开（竖屏纵向排列更佳）：
  - 节点1「公开搜索」→ 节点2「竞品官网」→ 节点3「sitemap/RSS」→ 节点4「Shopify 路径」
- 每个节点 spring 弹入 + 数据流连线动画（光点沿线流动）
- 节点亮起时弹出小标签：robots / sitemap / RSS / collection
- 底部计数器跳动：「发现 24 个候选 URL」

### 场景 4 · 九类数据库（14–20s，180 帧）
**文案**：自动建立九类行业数据库

**视觉**：
- 3×3 网格卡片逐个 spring 弹入（stagger，每张间隔 12 帧）
- 每张卡片：图标 + 库名 + 计数
  - 信息源 8 / 竞品 6 / 网站结构 3
  - 产品 6 / 关键词 6 / 痛点 5
  - 内容 5 / 机会 6 / 周报 1
- 卡片点亮时发出青色光晕，连线到中心"数据库"核心
- 中心核心脉冲呼吸
- 顶部标题：「九类数据库 · 一次建好」

### 场景 5 · 机会评分 + 报告（20–26s，180 帧）
**文案**：机会评分 · 人工审核 · Markdown 报告

**视觉**：
- 左侧大数字 count-up 动画：0 → 88（机会评分）
- 评分环弧形进度条填充（翠绿色）
- 右侧机会卡 spring 弹出：「换粮过渡期肠道套装 · 综合评分 88」
- 下方报告卡片生成动画：标题行 → 段落行逐行刷出（模拟 Markdown 渲染）
- 报告顶部标签：「已确认发现 / 证据不足但可能成立 / 阻塞项」
- 底部小字：「每条结论可溯源到 evidenceId + URL」

### 场景 6 · CTA 收尾（26–30s，120 帧）
**文案**：行业研究生产台 / CLI-first · 公开采集 · 人工审核

**视觉**：
- 全屏深色，中心光晕扩散
- 品牌名 spring 弹出：「行业研究生产台」
- 副标题淡入：「Industry Research Production Table」
- 三行特性标签逐行淡入上滑：
  - 「CLI-first 公开采集」
  - 「九库 + 机会评分」
  - 「人工审核 · 可溯源」
- 底部域名条：「research.playgamelab.cn」+ 数据流粒子向上下散开

## 资产需求

- 无外部图片资产，全部用 Remotion 代码 + Lucide 风格 SVG 绘制
- 字体走 `@remotion/google-fonts`（Space Grotesk + Noto Sans SC）
- 无 BGM（如需可后续追加免费商用音乐）

## 技术参数

- fps: 30
- 总帧数: 900
- 分辨率: 1080×1920
- 编码: H.264 MP4
- 转场: 场景间 15 帧交叉淡入（crossfade），无硬切
